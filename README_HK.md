## 📊 平台運行狀態 (Store Status)

### ✅ 核心服務 (Core Services)
* [x] **統一身份認證 (SSO)**: 一鍵 Login 晒所有 CSJSS 系列 App，唔使由頭入過 Account
* [x] **OTA 自動更新**: 當子 App 有新 Version，個 Store 會自動彈 Update 通知
* [x] **權限分級**: 自動認你是邊個（Login CSJSS Google account）（學生/領袖生/老師），開返對應嘅 App 下載權限
* [x] **跨平台支援**: 完美支援 iOS (IPA) 同 Android (APK) 環境（註：CSJSS APPS 喺 iOS 唔需要裝 App，Store 頁面會變做一個 App 整合包）
* [x] **Beta 測試版軟件分發**：可以搶先試玩仲開發緊嘅 Beta 版 App
* [x] **CSJSS Healthy Kit**：基於 EazyUI 自研架構同 MIT App Inventor 開發嘅智能健康監測 App，靠 iRED 公司整嘅 Smart Learning HEALTH KIT 硬件

## 💳 CSJSS PASS 核心亮點 (Spotlight)

**將你張學生證放入手機 Wallet，體驗「直行直過」嘅方便**

* **Wallet Integration (原生銀包整合)**:
    * 支援一撳加入 **Apple Wallet** (iPhone/Apple Watch)
    * 支援一撳加入 **Google Wallet** (Android)
* **Tap to Check-in (NFC 拍卡)**:
    * 朝早返學嗰陣，唔使解鎖手機或者開 App，直接 **NFC 拍卡** 就記到返學時間
    * 圖書館借書、Lab 簽到、小食部驗證身份，一拍就得
* **Security (安全驗證)**:
    * 動態加密 QR Code，杜絕 Cap 圖借證同盜用身份

## 🧪 CSJSS Lab 創新體驗 (Innovation)

**打破物理限制，將 Science Lab 放入褲袋**

1.  **安全守則資料庫**: 做實驗前必讀 Guide，內置測驗，Pass 咗先准入實體 Lab
2.  **AR 元素週期表**: 用手機鏡頭 Scan 書入面啲圖，3D 浮個化學元素出嚟睇
3.  **VR 虛擬實驗**: 模擬危險化學反應（好似鈉放入水、稀釋強酸），等同學喺安全環境下睇大爆炸

## ⚙️ 安裝與部署 (Deployment)

**因為呢個係校內專用 Store，麻煩跟住下面步驟裝：**

1.  **下載安裝包**:
    * 去校網 Download `CSJSS_Store_Installer` (iOS 描述檔 或 Android APK)。
2.  **信任企業憑證 (iOS)**:
    * 設定 -> 一般 -> VPN 與裝置管理 -> 㩒信任 `Caritas St. Joseph Secondary School`。
3.  **身份登入**:
    * 用學校 Google Account (`@csjss.edu.hk`) Login 個 Store，系統會自動派對應權限畀你。
4.  **一鍵部署**:
    * 㩒首頁個 "Install All Essential Apps" 就可以一次過 Download 晒必備工具。
